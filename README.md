# MyJobs_WeChat
微信小程序的的各种实战demo

在这里本人将以往做过的小程序相关的项目一一做了总结，并且拆分成一个个细小的功能模块（这里包括：UI界面搭建、数据操作、用户或数据交互逻辑），封装抽类成一个个独立的功能。会把涉及到的功能点和知识点做详细的注释，便于理解、学习、使用。

## 微信小程序--功能列表 </br>
1、下拉刷新，上拉加载
2、组件行切换时改变样式
3、封装提示框
4、扫码
5、选择器
6、定位
7、计时器
8、获取验证码
